Application Builder

Application Builder is designed with the intent of being a modular form creator for job postings. It alows an employer to create a responsive application that an applicant can fill out on the site.

Application Builder can be found here

This project was built with React as the framework, PostgreSQL for the database, and an Express Node server

The all listings page is open to public view for people to browse job offers. Upon signup/login they can fill out an application and submit it.

Upon signup you can distinguish if you are looking to appy for jobs or create listings. Company accounts still maintain full access to all standard account features.

Employers can view anyone who has applied for their posting. Each applicant is displayed with a percentage value to indicate how well they match the employers specifications as well as being color coded for convenience.

A fully modular application creator capable of taking a dynamic amount of questions and answers. Point values can be attributed to each question to help create a percentage match for each applicant.
