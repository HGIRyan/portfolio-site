notStackOverflow

notStackOverflow is a clone of the very popular StackOverflow. It is a group project created with the purpose of cloning the complex logic used by the original site.

notStackOverflow live site can be found here

notStackOverflow source can be found here

This project was built with React as the framework, PostgreSQL for the database, and an Express Node server

The main page displays most recent the questions along with votes, views and answers.

The tags section keeps a record of all subjects across the website. Clicking on one displays all questions related to the specified tag.

notStackOverflow question creator was created with lodash and quiljs, resulting in a clone capable of recording and storing the data the same as the original StackOverflow

All questions can be viewed. Upon viewing a question you can post answers, post comments, vote, favorite and edit. Questions are rendered from the database as html strings created by quiljs.

Editing lets you see all past revisions as well as pending edits. The owner of the question as well as moderators can accept edits to change the orignal posting.

The home pages can also be sorted by various subjects. This photo shows an example of the sort votes function.
